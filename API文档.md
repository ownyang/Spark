# 获取最近课程

##  借口描述

获得该用户最近完成的课程（10次），使用GET

## 请求参数

|  Key   |     Value      |
| :----: | :------------: |
| `wxid` | 微信id（字符） |



## 返回值

|    Key    |                           Value                           |
| :-------: | :-------------------------------------------------------: |
| `classes` | 列表[ 列表[时间，学生/老师名（字符），反馈状态（布尔）] ] |

# 新建老师

##  借口描述

新增一个老师的记录，使用POST

## 请求参数

|   Key    |                            Value                             |
| :------: | :----------------------------------------------------------: |
|  `wxid`  |                        微信id（字符）                        |
| `record` | 列表[姓名，性别，出生日期，电话，学校，年级，QQ号，休息时间，监护人姓名，<br>监护人职业，监护人电话，申请部门，是否出国] |

## 返回值

|    Key    |        Value         |
| :-------: | :------------------: |
| `success` | 是否成功上传（布尔） |
